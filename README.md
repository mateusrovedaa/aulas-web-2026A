# Desenvolvimento para a Internet — 2026/A

Repositório de aulas e exercícios da disciplina **Desenvolvimento para a Internet** do Curso Técnico da [Univates](https://www.univates.br/), semestre 2026/A.

**Professor:** Mateus Roveda

---

## Tecnologias

Ao longo da disciplina serão utilizadas as seguintes tecnologias:

| Tecnologia   | Descrição                              |
|--------------|----------------------------------------|
| HTML         | Estruturação de páginas web            |
| CSS          | Estilização e layout                   |
| JavaScript   | Interatividade no lado do cliente      |
| PHP          | Programação no lado do servidor        |
| PostgreSQL   | Banco de dados relacional              |

## Estrutura do Repositório

```
aulas-web-2026A/
├── 2402/       # Aula de 24/02
└── ...         # Demais aulas organizadas por data
```

## Como usar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Os arquivos HTML e CSS podem ser abertos diretamente no navegador. Para os exemplos com PHP e PostgreSQL, é necessário um servidor web com suporte a PHP e uma instância do PostgreSQL configurada.
